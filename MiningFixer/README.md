# MiningFixer

## Установка
### appsettings.json
* MaxAllowedPower: максимально допустимая суммарная мощность всех карт.
* Threshold: допустимое количество сигналов о превышении допустимой мощности, прежде чем будет вызван скрипт понижения напряжения.
* VoltageFixerBatFile: путь к bat-файлу устанавливающему напряжение.
* LogDirectory: путь к папке с логами майнера.

### Установка службы
* Запустить с правами администратора файл install.bat.

## Удаление
* Перед удалением запустить с правами администратора uninstall.bat.
